In order to enable security advisories in GitHub, you will first need to ensure you have the correct permissions - you will need to be an administrator of the repository or the organisation. Once it has been established that you have the correct permissions, follow the below instructions:

  1. Navigate to the main page of the repository
  2. Under the repoistory name, click settings
  3. Under "Data services", select Allow GitHub to perform read-only analysis of this repository
  4. Select the checkboxes under Dependency Graph and Vulnerability Alerts
