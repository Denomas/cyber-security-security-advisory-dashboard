Introduction

This tool captures and analyses a bunch of data about the repositories within the GitHub {{ content.org }} organisation.

The goal is to identify potential security improvements in terms of:

  • configuring active monitoring with GitHub security advisories
  • automated patching with Dependabot
  • identifying current known vulnerabilities
  • targeting high and critical severity patches
  • identifying un-archived and un-maintained repositories

The app has 2 primary pages

Overview

Shows statistics about the alphagov organisation.

By repository

Shows the individual repositories and their config.
